# prime-component

## Project setup
```
yarn install
```

### Build
```
开发：yarn build
测试：yarn build:test
预发：yarn build:pre
线上：yarn build:prod
```
### 本地启动环境
```
开发：yarn dev
测试：yarn test
预发：yarn pre
线上：yarn prod
```

### 单元测试
